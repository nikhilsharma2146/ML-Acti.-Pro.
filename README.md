High-fidelity Machine Learning visualization dashboard built with **Streamlit**, **Scikit-Learn**, and **Plotly**. Tailored for Gen Z with a premium Cyberpunk/Neon UI and Hinglish explanations.

## 📁 Project Structure

- `ml_genz_dashboard.py`: Main entry point and UI logic.
- `styles.py`: Custom CSS, floating animations, and Plotly theme.
- `data_utils.py`: Synthetic data generation with caching.
- `model_utils.py`: Model initialization and real-world use case data.
- `requirements.txt`: List of dependencies.

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Dashboard:**
   ```bash
   streamlit run ml_genz_dashboard.py
   ```

## ✨ Features

- **Dynamic Data Gen:** Sliders for samples, features, and noise.
- **Multiple Models:** Logistic Regression, Decision Trees, Random Forest, and KNN.
- **Interactive Plots:** Plotly-powered scatter plots, confusion matrices, and ROC curves.
- **Model Showdown:** Compare all models (Acc, Precision, F1) with one click.
- **Export & Save:** Download trained models (.joblib) and datasets (.csv).
- **Hinglish Explanations:** ML concepts explained in relatable language with emojis.
- **Premium UI:** Glassmorphism, neon glows, and smooth animations.
